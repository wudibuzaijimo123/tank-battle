# 坦克大战 🎮 Battle City

经典坦克大战游戏的 HTML/JavaScript 实现，单文件即开即玩。

## 快速开始

直接用浏览器打开 `index.html`，或访问 GitHub Pages 链接。

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → 或 W A S D | 移动坦克 |
| 空格 (Space) | 开火 |
| P | 暂停 / 继续 |

## 游戏规则

- 消灭所有敌方坦克即可过关
- 保护基地（右下角的⭐）不被摧毁
- 每关有 20+ 辆敌方坦克，分波次出现
- 共有 3 个关卡

## 道具

击毁敌方坦克有概率掉落道具：

| 道具 | 效果 |
|------|------|
| 🛡 护盾 | 一段时间内无敌 |
| ❤ 生命 | 增加一条命 |
| ⚡ 加速 | 短时间内提升移动速度 |

## 特性

- 3 个独特关卡地图
- 敌人 AI（追踪、射击、随机移动）
- 可破坏砖墙 + 不可破坏钢墙
- 粒子爆炸效果
- 出生保护
- 关卡递进

## 技术栈

纯 HTML + CSS + JavaScript，无任何外部依赖。
